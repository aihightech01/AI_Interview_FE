AI# 🧠 AI 면접 코칭 서비스 - Frontend

> 실전처럼 연습하고, AI 피드백으로 성장하는 **AI 면접 코칭 웹 서비스**의 프론트엔드 레포지토리입니다.

---

## 📌 프로젝트 소개

**AI 면접 코칭 서비스**는 면접 준비를 하는 사용자가  
웹캠과 마이크를 통해 **혼자서도 실전처럼 면접을 연습**하고,  
AI가 **답변 내용·표정·시선·말 속도** 등을 분석해 피드백을 제공하는 서비스입니다.

이 레포지토리는 해당 서비스의 **프론트엔드(React 기반)** 코드와  
UI/UX 구현을 담당하고 있습니다.

---

## 🎯 주요 기능

- 🎙 **면접 세션 진행**
  - 질문 선택 / 직무 선택 / 난이도 선택
  - 웹캠 + 마이크로 답변 녹화 (실시간 타이머, 진행 상태 표시)

- 🤖 **AI 분석 결과 보기**
  - 답변 내용 요약 & 부족한 부분 코멘트
  - 말 속도 / 시선 / 표정 등 비언어적 요소 피드백 (텍스트 기준)

- 📋 **세션 리포트**
  - 질문별 점수 / 피드백 목록
  - 다음 연습을 위한 개선 포인트 정리

- 👤 **사용자 흐름**
  - 홈 → 인터뷰 (자소서 기반) 질문 생성 → 기기셋팅 / 카메라 조정 → 면접 진행 → 분석 대기 → 결과 리포트

> ※ 실제 AI 분석은 **백엔드 / 외부 API**( Gemini)를  연동하여 수행하며, 이 레포는 **프론트 화면 & 호출 부분**을 담당

---

## 🛠 기술 스택

### **Frontend**

* ⚛️ **React 19** 
* ⚡ **Vite** 
* 🧭 **React Router** 
* 🎣 **Custom React Hooks**
* 🎨 **Tailwind CSS** 
* 🗂 **Zustand** — 인증·인터뷰 데이터 전역 상태 관리 (`authStore`, `interviewStore`)
* 🔁 **Axios** — 서버 및 Mock API 통신
* 🧪 **MSW (Mock Service Worker)** — 백엔드 없이 MockAPI 초기 개발 
* 🎥 **Media API** — 카메라/마이크 접근 및 제어 (`media.js`)
* 📊 **데이터 전처리 유틸**

  * `seriesNormalize`, `transformEmotion`, `provisionalCount` 등 그래프/감정 분석용 처리
